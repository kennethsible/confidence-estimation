Interactive Translator
  • Enter a sentence in the input field (top/left) and click the arrow button to translate.
  • If the arrow symbol changes to an exclamation, the translation API is unreachable.
  • Words that may have been mistranslated are highlighted in blue in the input field.
  • Click on a highlighted word to see suggestions that may improve the translation.
  • If an input word has an orange highlight, it is out-of-vocabulary for suggestions.
  • To change/update the input sentence, double-click anywhere in the input field.
  • Open the JavaScript console to see uncertainty scores for each word in the input.
  • Use "Add to Home Screen" on Android/iOS to add an icon for the standalone app.
